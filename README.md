# Gather

A privacy-focused event planning app for creating parties and gatherings.

## What is this?

Gather lets you create events, invite friends, and coordinate without sacrificing privacy. No tracking, no ads, minimal data collection.

## Features

- **Low-friction signup** - Phone number + name, that's it
- **Simple event creation** - Title, description, when, where
- **Easy invitations** - Share a link or send via email/SMS
- **RSVP tracking** - Yes, No, Maybe with optional questionnaires
- **Event wall** - Attendees can chat and coordinate
- **Connections** - Remember people you've partied with
